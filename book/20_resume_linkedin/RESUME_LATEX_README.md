# LaTeX resume template (AI Data Engineer)

The file **`ai_data_engineer_resume.tex`** lives in this folder (`book/20_resume_linkedin/`) — not in a subfolder — so you can download or copy it directly.

## File

| File | Purpose |
|------|---------|
| `ai_data_engineer_resume.tex` | Main template — **replace all `% TODO:`** placeholders with your details |

## Edit online (recommended)

**[Overleaf](https://www.overleaf.com/)** — free account, no local install:

1. Go to [https://www.overleaf.com](https://www.overleaf.com)
2. **New Project** → **Upload Project** → upload `ai_data_engineer_resume.tex` (or paste into a blank project)
3. Click **Recompile** to produce PDF
4. **Menu → Download → PDF**

Docs: [https://www.overleaf.com/learn](https://www.overleaf.com/learn)

## Edit locally

```bash
cd book/20_resume_linkedin
latexmk -pdf -interaction=nonstopmode ai_data_engineer_resume.tex
```

## ATS tips

* **One column** (this template is single-column).
* **Standard** section names: Profile, Technical Skills, Experience, Education.
* Export **PDF**; verify text is **selectable** (not image-only).
